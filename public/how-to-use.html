How to Use WhatsApp Bulk Sender

Follow these steps to successfully send bulk messages

1

Enter Phone Numbers Correctly

For Pakistan numbers, use one of these formats:

923422187767
03422187767
3422187767
Important: Do NOT include + sign or spaces. Just use digits only.
2

Add Contacts

You can add contacts in two ways:

  1. Manual Entry: Go to "Contacts" tab → "Add Contacts Manually" → Enter phone number and name (optional)
  2. CSV Upload: Create a CSV file with format: phone_number,name → Upload in "Upload Contacts" section
923422187767,John Doe
923331234567,Jane Smith
3

Write Your Message

Go to the "Dashboard" tab and:

  1. Enter your message in the text area
  2. Set delay between messages (15 seconds recommended to avoid blocking)
Tip: Keep messages concise. WhatsApp may block you for sending too many messages too quickly.
4

Start Sending

Click the "Start Sending" button. A progress modal will appear showing:

  • Progress bar with percentage
  • Current contact being messaged
  • Status of sending process

You can click "Stop Sending" at any time to pause the process.

5

Check Results

After sending, you can view:

  • Dashboard: Statistics on total contacts, messages sent, and pending
  • Contacts Tab: Status of each contact (sent, failed, pending)
  • History Tab: Complete sending history with timestamps
Troubleshooting
Messages failing to send?
  • Check phone number format (digits only, no + or spaces)
  • Ensure WhatsApp is connected (check QR code section)
  • Try adding country code manually (92 for Pakistan)
  • Check if you have the contact saved in your WhatsApp
  • Wait a few minutes and try again (WhatsApp may temporarily block)
Best Practices
  • Use delay of at least 15 seconds between messages
  • Don't send to too many contacts at once (start with 10-20)
  • Personalize messages when possible
  • Avoid sending spam or promotional content
  • Monitor the sending process and stop if you see errors
